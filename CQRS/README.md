# CQRS (Command and Query Responsibility Segregation)
```md
CQRS就是对事务脚本和领域模型两种模式的综合
因为对于Query和报表的场景，使用领域模型往往会把简单的事情弄复杂，
此时完全可以用奥卡姆剃刀把领域层剃掉，直接访问Infrastructure。
```
